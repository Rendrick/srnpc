

## Sistema NPS - Clínica de Radiologia

### 1. Página de Pesquisa NPS (pública)
- Pergunta principal: "De 0 a 10, qual a probabilidade de recomendar nossa clínica?"
- Botões de 0 a 10 com cores (vermelho → amarelo → verde)
- Campo de comentário aberto após selecionar a nota
- Tela de agradecimento após envio
- Link compartilhável (ex: `/pesquisa`) para enviar por WhatsApp ou gerar QR Code

### 2. Dashboard Administrativo
- **Score NPS geral** em destaque (gauge/número grande com cor)
- **Gráfico de evolução** do NPS ao longo do tempo (linha)
- **Distribuição**: barras/donut mostrando % Detratores (0-6), Neutros (7-8) e Promotores (9-10)
- **Total de respostas** e média de notas
- Cards resumo com ícones

### 3. Gestão de Respostas
- Tabela com todas as respostas (nota, comentário, data, tipo de exame)
- Filtros por período, nota e categoria (detrator/neutro/promotor)
- Badges coloridos para classificação
- Busca por texto nos comentários

### 4. Relatórios por Período/Setor
- Filtros de data (últimos 7/30/90 dias ou personalizado)
- Filtro por tipo de exame (Raio-X, Tomografia, Ressonância, Ultrassom, Mamografia)
- Gráficos comparativos entre períodos
- Exportação visual dos dados

### 5. Navegação
- Sidebar com menu: Dashboard, Respostas, Relatórios, Nova Pesquisa
- Logo/nome da clínica no topo
- Design limpo e profissional com tons de azul (remetendo à saúde)

> **Nota**: Os dados serão mockados para demonstração, sem banco de dados. Todas as funcionalidades estarão visuais e interativas com dados de exemplo.
