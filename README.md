# NPS Service Rapide

Pesquisa de satisfação NPS (Net Promoter Score).

## Como executar

Requisitos: Node.js e npm instalados.

```sh
# Instalar dependências
npm i

# Servidor de desenvolvimento
npm run dev

# Build para produção
npm run build

# Testes
npm run test

# Lint
npm run lint
```

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
